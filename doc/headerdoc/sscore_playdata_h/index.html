Introduction

The C interface to play information in SeeScoreLib

The barindex parameter is zero-based. ie 0 is the first bar in the score. All times are in milliseconds

sscore_pd_loopbegin

get an iterator which goes repeatedly from startBarIndex to loopBarIndex for the given number of repeats 

EXPORT sscore_pd_bar_iterator sscore_pd_loopbegin(
    const sscore_playdata *pd,
    int startBarIndex,
    int loopBarIndex,
    int numRepeats);
Parameters
pd

the playdata

startBarIndex

the index for the bar to loop from and return to at the start of each loop

loopBarIndex

the index for the last bar to play in the loop

numRepeats

the number of loops

Return Value

    unsigned dummy[4]; // future
